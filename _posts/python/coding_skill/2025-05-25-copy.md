# 참조와 복사

- 참조(복사 X)
- 깊은 복사
- 얕은 복사

## 참조

이건 복사가 아니다. 새 객체를 만들지도 않고, 같은 메모리 주소를 공유한다.

```python
a = [1, 2, 3]
b = a        # b는 a와 같은 객체를 가리킴

b[0] = 99
print(a)     # [99, 2, 3] ← a도 바뀜!
print(a is b)  # True (같은 객체)
```

## 얕은 복사

최상위 객체는 새 객체로 만들지만, 내부 중첩 객체는 여전히 참조를 공유한다.

```python
import copy

a = [1, [2, 3], 4]
b = copy.copy(a)      # 또는 a[:], list(a)

b[0] = 99
print(a)  # [1, [2, 3], 4] ← 바뀌지 않음

b[1][0] = 99
print(a)  # [1, [99, 3], 4] ← 중첩 객체는 바뀜
```

## 깊은 복사

모든 중첩 객체까지 재귀적으로 새로 만들어 완전히 독립된 복사본을 만든다.

```python
import copy

a = [1, [2, 3], 4]
b = copy.deepcopy(a)

b[1][0] = 99
print(a)  # [1, [2, 3], 4] ← 전혀 영향 없음 ✅
```

## 한눈에 비교

| 구분      | 새 객체 생성 | 중첩 객체 공유 | 완전 독립 |
| --------- | ------------ | -------------- | --------- |
| `b = a`   | ❌           | ❌ (완전 공유) | ❌        |
| 얕은 복사 | ✅           | ⚠️ (공유됨)    | ❌        |
| 깊은 복사 | ✅           | ✅ (독립 복사) | ✅        |
