### 브루트 포스 계산

배열을 2번 반복하면서 모든 조합을 더해서 일일이 확인해보는 무차별 대입 방식

### 딕셔너리를 이용한 풀이

1. 배열을 1번만 반복하면서, 요소들을 target과 뺀다.
2. 그 값이 딕셔너리에 있다면, 요소의 인덱스와 해당 값에 대한 인덱스를 출력한다.
3. 없다면, 요소를 key로 인덱스를 value로 딕셔너리에 저장한다.

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        nums_map = {}
        # 1
        for ind,num in enumerate(nums):
            if target-num in nums_map:
                # 2
                return [nums_map[target-num], ind]
            # 3
            nums_map[num] = ind
```

### Two-pointer

### Two-Pointer

낮은 곳에서 높은 곳으로 이동 한다.

1. 좌우 높은 위치를 확인하고, 더 낮은 곳을 움직인다.
2. 현재 높이와 이전 제일 높은(또는 현재 높이)와 빼서 volume을 추가

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        # two pointer
        volume = 0
        left, right = 0, len(height)-1
        left_max, right_max = height[left], height[right]

        while left < right:
            # 현재 높이와 이전 제일 높은 높이 비교
            left_max, right_max = max(left_max, height[left]), max(right_max,height[right])

            if left_max < right_max:
                volume += left_max - height[left]
                left +=1

            elif left_max >= right_max:
                volume += right_max - height[right]
                right -=1

        return volume
```

### Stack

1. 변곡점을 찾는다. (현재 높이가 이전 높이 보다 높을 때 (변곡점))
2. 변곡점을 만나기 전까지 stack에 쌓는다.
3. distance(index)과 height 을 찾아 Volume을 구한다.

3번 부분을 계산하기 어려웟다.

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        # stack
        stack = []
        volume = 0
        for pos, hei in enumerate(height):
        # Inflection Point - 변곡점
            while stack and height[stack[-1]] < hei:
                # volume 을 구하는 공식
                prev_pos = stack.pop() # 물이 쌓일 지점
                # 물이 쌓일 수 없을 때
                if len(stack) == 0:
                    break

                dist = (pos - stack[-1]) -1
                water_height = min(hei, height[stack[-1]]) - height[prev_pos]

                volume += dist * water_height

            # 현재 높이가 변곡점이 아니라면, stack에 추가한다.
            stack.append(pos)
        return volume
```

- 놓친 부분 : while 문 조건에서 stack 을 빼먹어서, hegith[stack[-1]] 부분에서 list index out of range 오류가 발생

## 풀이

### 브루트 포스

### Two Pointer

- Two-Pointer를 적용하기 위해 정렬
- 요소를 반복하면서 요소를 제외한 나머지를 Two-pointer로 조회 한다.
- 3요소의 합이 0인 것을 찾는다.
- 중복 인덱스, 중복 요소의 합은 빼줘야 한다.(중복 제거)

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        # 종료 조건
        result = []
        if len(nums) == 3:
            if sum(nums) == 0:
                result.append(nums)
            return result

        # two pointer
        nums.sort()
        for ind,num in enumerate(nums):
            # 중복 제거
            if ind != 0 and num == nums[ind-1]:
                continue

            remain = nums[ind+1:]
            left, right = 0, len(remain) -1

            while left < right:
                if (s:=num + remain[left] + remain[right]) > 0 :
                    right -= 1
                elif s < 0:
                    left +=1
                else:
                    result.append([num,remain[left],remain[right]])
                    # 중복 제거
                    while left < right and remain[left] == remain[left+1]:
                        left +=1
                    while left < right and remain[right] == remain[right-1]:
                        right -=1
                    left +=1
                    right -=1
        return result

```

- 놓친 부분 : 중복 제거 - for 문에서의 중복

# LeetCode 561

n개의 페어를 이용한 min(a,b)의 합으로 만들 수 있는 가장 큰 수를 반환하라.

## 풀이

### 오름차순 풀이

반복문을 사용해서 두가지 요소마다 min()을 적용하여, `result += min(a,b)`를 적용한다.

```python
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        s = 0
        pair = []
        nums.sort()

        for n in nums:
            pair.append(n)

            if len(pair) == 2:
                s += min(pair)
                pair = []
        return s
```

### 짝수 번째 값 계산

min(a,b)는 정렬을 하고 나면 앞에 있는 값이다. 그리고 그 값들이 두칸씩 떨어져 있다.

이 내용을 이용하여 파이썬 함수를 사용해서 쉽게 풀 수 있다.

```python
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        return sum(sorted(nums)[::2])
```

<p>&nbsp;</p>

## 풀이

나눗셈을 이용하지 않고 O(n)에 풀이해야 한다.
미리 전체 곱셈 값을 구해 놓고 각 항목별로 자기 자신을 나눠서 풀이하는 방법은 안된다.

(왼쪽 곱한 값 \_ 오른쪽 값) 을 통해 각 요소를 제외한 모든 요소의 값을 구한다.

$$3 : (1 * 2) * (4) = 8$$

왼쪽 값 구하기 1을 먼저 채우고 요소 하나씩 곱해서 다음 요소로 넘어간다.
`[1,1,2,6]`
오른쪽은 반대로 4부터 시작한다.`[1,4,12,24]`의 역순 `[24,12,4,1]`
각 요소를 곱해준다.

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        # 좌우 곱셈 결과
        def products(nums,sub=1):
            for num in nums:
                yield sub
                sub *= num
        result = []
        left = products(nums)
        right = list(products(nums[::-1]))[::-1]

        for ln, rn in zip(left,right):
            result.append(ln * rn)
        return result
```

# Leetcode 121

한 번의 거래로 낼 수 있는 최대 이익을 산출하라.

## 풀이

1. 요소를 반복 할 때 마다. min()함수를 사용해서 최소 값을 갱신한다.
2. 최소 값을 이용해서, 최대 이익을 얻을 수 있게 max()함수를 이용한다.

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_num = sys.maxsize
        profit = -sys.maxsize
        if not profit:
            return 0

        for p in prices:
            min_num = min(p,min_num)
            profit = max(profit, p - min_num)

        return profit

```
