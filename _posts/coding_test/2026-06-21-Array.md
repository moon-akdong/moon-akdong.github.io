# Leetcode 1

<hr><p>Given an array of integers <code>nums</code>&nbsp;and an integer <code>target</code>, return <em>indices of the two numbers such that they add up to <code>target</code></em>.</p>

<p>You may assume that each input would have <strong><em>exactly</em> one solution</strong>, and you may not use the <em>same</em> element twice.</p>

<p>You can return the answer in any order.</p>

<p>&nbsp;</p>
<p><strong class="example">Example 1:</strong></p>

<pre>
<strong>Input:</strong> nums = [2,7,11,15], target = 9
<strong>Output:</strong> [0,1]
<strong>Explanation:</strong> Because nums[0] + nums[1] == 9, we return [0, 1].
</pre>

<p><strong class="example">Example 2:</strong></p>

<pre>
<strong>Input:</strong> nums = [3,2,4], target = 6
<strong>Output:</strong> [1,2]
</pre>

<p><strong class="example">Example 3:</strong></p>

<pre>
<strong>Input:</strong> nums = [3,3], target = 6
<strong>Output:</strong> [0,1]
</pre>

<p>&nbsp;</p>
<p><strong>Constraints:</strong></p>

<ul>
	<li><code>2 &lt;= nums.length &lt;= 10<sup>4</sup></code></li>
	<li><code>-10<sup>9</sup> &lt;= nums[i] &lt;= 10<sup>9</sup></code></li>
	<li><code>-10<sup>9</sup> &lt;= target &lt;= 10<sup>9</sup></code></li>
	<li><strong>Only one valid answer exists.</strong></li>
</ul>

<p>&nbsp;</p>
<strong>Follow-up:&nbsp;</strong>Can you come up with an algorithm that is less than <code>O(n<sup>2</sup>)</code><font face="monospace">&nbsp;</font>time complexity?

## 풀이

### 브루트 포스 계산

배열을 2번 반복하면서 모든 조합을 더해서 일일이 확인해보는 무차별 대입 방식

### 딕셔너리를 이용한 풀이

1. 배열을 1번만 반복하면서, 요소들을 target과 뺀다.
2. 그 값이 딕셔너리에 있다면, 요소의 인덱스와 해당 값에 대한 인덱스를 출력한다.
3. 없다면, 요소를 key로 인덱스를 value로 딕셔너리에 저장한다.

````python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        nums_map = {}
        # 1
        for ind,num in enumerate(nums):
            if target-num in nums_map:
                # 2
                return [nums_map[target-num], ind]
            # 3
            nums_map[num] = ind
```

### Two-pointer

인덱스를 출력으로 받기 때문에 정렬을 할 수 없다.
여기서 Two-pointer는 작으면 왼쪽 pointer를 옮기고, 크다면 오른쪽 pointer를 옮겨야 한다. 하지만 정렬이 되어 있지 않기 때문에, 적용을 할 수 없다.

## Two-Pointer

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        # two pointer
        volume = 0
        left, right = 0, len(height)-1
        left_max, right_max = height[left], height[right]

        while left < right:
            # 현재 높이와 이전 제일 높은 높이 비교
            left_max, right_max = max(left_max, height[left]), max(right_max,height[right])

            if left_max < right_max:
                volume += left_max - height[left]
                left +=1

            elif left_max >= right_max:
                volume += right_max - height[right]
                right -=1

        return volume
````

## Stack

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        # stack
        stack = []
        volume = 0
        for pos, hei in enumerate(height):
        # Inflection Point - 변곡점
            while stack and height[stack[-1]] < hei:
                # volume 을 구하는 공식
                prev_pos = stack.pop()

                if len(stack) == 0:
                    break

                dist = (pos - stack[-1]) -1
                water_height = min(hei, height[stack[-1]]) - height[prev_pos]

                volume += dist * water_height

            # stack 쌓기
            stack.append(pos)
        return volume
```

- 놓친 부분 : while 문 조건에서 stack 을 빼먹어서, hegith[stack[-1]] 부분에서 list index out of range 오류가 발생

## Two Pointer

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        # 종료 조건
        result = []
        if len(nums) == 3:
            if sum(nums) == 0:
                result.append(nums)
            return result

        # two pointer
        nums.sort()
        for ind,num in enumerate(nums):
            # 중복 제거
            if ind != 0 and num == nums[ind-1]:
                continue

            remain = nums[ind+1:]
            left, right = 0, len(remain) -1

            while left < right:
                if (s:=num + remain[left] + remain[right]) > 0 :
                    right -= 1
                elif s < 0:
                    left +=1
                else:
                    result.append([num,remain[left],remain[right]])
                    # 중복 제거
                    while left < right and remain[left] == remain[left+1]:
                        left +=1
                    while left < right and remain[right] == remain[right-1]:
                        right -=1
                    left +=1
                    right -=1
        return result

```

- 놓친 부분 : 중복 제거 - for 문에서의 중복

```python
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        s = 0
        pair = []
        nums.sort()

        for n in nums:
            pair.append(n)

            if len(pair) == 2:
                s += min(pair)
                pair = []
        return s
```

# Leetcode 238

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        # 좌우 곱셈 결과
        def products(nums,sub=1):
            for num in nums:
                yield sub
                sub *= num
        result = []
        left = products(nums)
        right = list(products(nums[::-1]))[::-1]

        for ln, rn in zip(left,right):
            result.append(ln * rn)
        return result
```

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_num = sys.maxsize
        profit = -sys.maxsize
        if not profit:
            return 0

        for p in prices:
            min_num = min(p,min_num)
            profit = max(profit, p - min_num)
```
